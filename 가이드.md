# 공시가격 조회 사이트 — 사용·배포 가이드

이 문서는 개발자가 아니어도 따라 할 수 있도록 단계별로 작성됐습니다. 총 소요시간은 약 **30~60분**입니다.

---

## 전체 흐름 요약

```
[1] Node.js 설치   →   [2] API 키 2개 발급   →   [3] 로컬에서 실행
       ↓
[4] GitHub에 올리기   →   [5] Vercel에 배포   →   [6] 완성!
```

---

## 1단계 ─ Node.js 설치 (10분)

프로젝트를 실행하려면 Node.js가 필요합니다.

1. https://nodejs.org 접속
2. **LTS 버전 (예: 20.x)** 다운로드 → 설치
3. 설치 후 터미널(명령 프롬프트 / PowerShell)에서 확인:
   ```
   node --version
   ```
   `v20.x.x` 처럼 표시되면 OK.

---

## 2단계 ─ API 키 2개 발급 (약 20분)

### 2-1. 공공데이터포털 키 발급

1. https://www.data.go.kr 접속 → 회원가입 (간편 인증 가능)
2. 로그인 후 아래 **3개 API**를 검색해서 각각 **활용신청**:

| API 이름 | 용도 |
|---------|-----|
| 국토교통부_부동산종합정보_개별주택가격 조회 | 단독/다가구 |
| 국토교통부_부동산종합정보_공동주택가격 조회 | 아파트/빌라 |
| 국토교통부_부동산종합정보_개별공시지가 조회 | 토지 |

3. **활용신청 시 작성 팁**
   - 활용 목적: "개인 학습용 경매 물건 분석"
   - 활용 형태: 웹 애플리케이션
   - 일일 트래픽: 1,000건 (기본)
4. 신청 즉시 또는 1영업일 내 승인됩니다.
5. **마이페이지 → 개발계정** 메뉴에서 **일반 인증키(Encoding)** 복사 → 메모장에 보관.

> 💡 3개 API 모두 같은 서비스 키 하나로 동작합니다.

### 2-2. VWorld (브이월드) 키 발급

지번주소를 법정동코드로 변환하는 용도입니다.

1. https://www.vworld.kr 접속 → 회원가입
2. **오픈API → 인증키 발급** 메뉴로 이동
3. 다음과 같이 신청:
   - 인증키 이름: `공시가격조회`
   - 서비스 URL: `http://localhost:3000` (로컬 테스트용)
     - 나중에 Vercel 배포 주소도 추가로 등록 가능
   - 서비스 분류: **geocoder API**
4. 신청 후 즉시 인증키 발급 → 복사해서 메모장에 보관.

## 3단계 ─ 로컬에서 실행 (10분)

### 3-1. 프로젝트 파일 준비

1. 이 폴더(`gongsi-lookup`)를 원하는 위치에 복사
2. 터미널을 열고 해당 폴더로 이동:
   ```
   cd C:\경로\gongsi-lookup
   ```

### 3-2. 패키지 설치

```
npm install
```

### 3-3. 환경변수 파일 생성

1. 폴더 안 `.env.example` 파일을 복사해서 `.env.local` 파일 생성
2. `.env.local` 을 메모장으로 열고 아래처럼 수정:

   ```
   DATA_GO_KR_KEY=여기에_공공데이터포털_서비스키_붙여넣기
   VWORLD_KEY=여기에_VWorld_인증키_붙여넣기
   MOCK_MODE=false
   ```

> 💡 API 키가 아직 없다면 `MOCK_MODE=true`로 두세요. 샘플 데이터로 UI를 먼저 테스트할 수 있습니다.

### 3-4. 로컬 서버 실행

```
npx vercel dev
```

- 첫 실행 시 Vercel 로그인을 요구합니다. 이메일로 간편 로그인하세요.
- 프로젝트 연결 질문이 나오면 **"Link to existing project? N"** → **"What's the name of your existing project? (엔터로 새로 생성)"** 순으로 진행.
- `http://localhost:3000` 이 열리면 **주소를 붙여넣어 테스트!**

### 3-5. 테스트 주소 예시

```
서울특별시 금천구 독산동 378-33 문정파크 제2동 제203호
서울특별시 강서구 화곡동 875-11 제4층 제401호
서울특별시 관악구 봉천동 1529-16
```

각각 붙여넣어보고 결과가 잘 나오는지 확인하세요.

## 4단계 ─ GitHub에 올리기 (10분, 선택)

Vercel 배포를 위해 GitHub에 코드를 올리면 편합니다. 이미 Git을 써보셨다면 바로 4-2로.

### 4-1. GitHub 준비

1. https://github.com 가입
2. 로그인 후 우측 상단 **+** → **New repository**
   - Repository name: `gongsi-lookup`
   - **Private** 선택 권장 (API 키가 노출되지 않도록)
3. **Create repository** 클릭

### 4-2. 터미널에서 업로드

프로젝트 폴더에서:

```
git init
git add .
git commit -m "첫 커밋"
git branch -M main
git remote add origin https://github.com/사용자명/gongsi-lookup.git
git push -u origin main
```

> ⚠️ **확인**: `.env.local` 파일은 절대 올라가면 안 됩니다. `.gitignore`에 이미 등록돼 있으니 `git status`로 파일 목록에 없는지 확인하세요.

---

## 5단계 ─ Vercel에 배포 (10분)

### 5-1. Vercel 연결

1. https://vercel.com 접속 → GitHub 계정으로 로그인
2. **Add New... → Project** 클릭
3. 방금 만든 `gongsi-lookup` 저장소를 Import

### 5-2. 환경변수 설정 (가장 중요!)

Import 화면에서 **Environment Variables** 섹션을 펼치고 3개 추가:

| Key | Value |
|-----|-------|
| `DATA_GO_KR_KEY` | (공공데이터포털 키) |
| `VWORLD_KEY` | (VWorld 키) |
| `MOCK_MODE` | `false` |

### 5-3. Deploy 클릭

약 1~2분 후 배포 완료. `https://gongsi-lookup-xxxx.vercel.app` 같은 주소가 생성됩니다.

### 5-4. VWorld 서비스 URL 업데이트

1. 발급받은 Vercel 주소(`https://gongsi-lookup-xxxx.vercel.app`)를 VWorld 인증키 관리 화면에서 **서비스 URL**에 추가 등록.
2. 이게 빠지면 VWorld가 호출을 차단합니다.

## 🛠 문제 해결

### "주소 형식을 인식하지 못했습니다"
- 지번주소 형식인지 확인 (도로명주소는 아직 미지원)
- "서울특별시"처럼 시/도명이 정식 명칭인지 확인

### "해당 주소의 법정동코드를 찾지 못했습니다"
- VWorld API가 응답하지 않는 경우. 주소를 다시 확인하거나 잠시 후 재시도.

### "공시가격 정보를 찾지 못했습니다"
- 신축 건물·미등재 필지·집합건물 전환 중인 경우
- 공동주택가격은 4~5월에 공시되니, 그 전에는 전년도 데이터 기준

### API 일일 호출 한도 초과
- 공공데이터포털 개발계정은 일일 1,000건 제한
- 마이페이지에서 **운영 활용신청**으로 전환하면 증량 가능

---

## ⚠️ 법적 주의사항

- 본 프로젝트가 제공하는 공시가격은 **참고용**입니다. 법적 효력은 국토교통부 공식 고시에 따릅니다.
- 공공데이터 API 이용약관상 **출처 표시 의무**가 있어 화면에 이미 명시돼 있습니다.
- 데이터를 대량 수집·재배포하면 이용약관 위반이 될 수 있습니다. 실시간 조회 용도로만 사용하세요.
- 경매 입찰 의사결정의 최종 책임은 본인에게 있으며, 이 서비스로 인한 손실에 대해 제작자는 책임지지 않습니다.
